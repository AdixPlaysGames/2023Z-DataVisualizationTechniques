Określenie problemu

Wydawało mi się, że nazwy specjalności na tym wykresie są nieczytelne oraz jest wielka różnica pomiędzy wielkimi i małymi znaczeniami, co powoduje mniejszą czytelność.

Dane

read.csv("ds_salaries.csv") -> df

library("dplyr")
library("ggplot2")
library("forcats")

df %>%
  group_by(job_title) %>%
  summarise(count = n()) %>%
  mutate(job_title = fct_reorder(job_title, count), logg = log10(count)) -> d

brks <- c(5, 10, 20, 40, 100, 150)

Poprawienie wizualizacji

Zdecydowałem zamienić osi miejscami, żeby łatwo można było przeczytać nazwy, oraz za pomocą scale_x_sqrt zmieniłem skalację oraz dodałem znaczenia count na wykresie, żeby łatwiej było wyciągać informację z niego. Dodatkowo dodałem gradient do kolumn, który jest ciemniejszy, o ile ilość prac jest większa.

ggplot(data = d, mapping = aes(y = job_title, x = count, fill = logg)) +
  geom_col() +
  scale_fill_gradient(low = "#00f887", high = "#43b581") +
  labs(title = "Number of Jobs", x = "Count (sqrt_scale)", y = "Job Title") +
  scale_x_sqrt(expand = c(0, 0, 0, 1), breaks = brks) +
  geom_text(aes(label = count), vjust = 0.3, nudge_x = 0.17, size = 3.2) +
  theme_bw() +
  theme(plot.title = element_text(hjust = 0.5), legend.position = "none", axis.text.y = element_text(size = 10))
